# UX Design Document

## Overview

The application will provide a simple interface where users can submit research queries and receive structured research reports generated by multiple AI agents.

The interface will be built using Streamlit.

---

## Main Screen Layout

The application will have the following sections:

1. Title Section
2. Query Input Section
3. Run Research Button
4. Research Results Section
5. Retrieved Evidence Section
6. Agent Activity Logs

---

## 1. Title Section

Displayed at the top of the page.

Example:

Autonomous Multi-Agent Research Assistant

This section helps users understand what the application does.

---

## 2. Query Input Section

Users will enter their research question here.

Example input box:

"Enter your research question"

Example query:

"What are the impacts of artificial intelligence in healthcare?"

---

## 3. Run Research Button

A button that starts the research process.

Example:

Run Research

When clicked, the system will trigger the multi-agent pipeline.

---

## 4. Research Results Section

This section will display the final structured report.

The report may include:

* Research Question
* Executive Summary
* Key Findings
* Evidence
* References

---

## 5. Retrieved Evidence Section

This section will display the pieces of information retrieved from the vector database.

Users can see the evidence used by the system.

---

## 6. Agent Activity Logs

This section shows what each agent is doing.

Example:

Planner Agent: Breaking query into subtasks
Retriever Agent: Searching vector database
Analyst Agent: Summarizing findings
Writer Agent: Generating final report
